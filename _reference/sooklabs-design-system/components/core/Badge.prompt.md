Compact status pill in mono type — use for system states and statuses (Connected, Assigned, Booked, Exposed, In progress). Tinted-glass fills keep it calm.

```jsx
<Badge tone="success" dot>Resolved</Badge>
<Badge tone="cyan">Connected</Badge>
<Badge tone="warning" dot>Assigned</Badge>
<Badge tone="danger">3 exposures</Badge>
```

Tones: `neutral | cyan | success | warning | danger | violet | blue`. Set `dot` for a leading glowing status dot. For category/topic labels prefer `Tag` instead.
