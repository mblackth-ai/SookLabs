Calm metric cell — mono value, optional delta and caption. Use in dashboards, SEOS snapshots, and proof-of-work strips. Keep numbers real and modest (the honesty motif).

```jsx
<StatTile label="Organic clicks" value="1,240" delta="+12%" caption="Month over month" />
<StatTile value="3 systems" caption="Sookly, SEOS, RoastMyOpSec" align="center" />
```

`deltaTone`: success / warning / danger / info / neutral. Lay several in a CSS grid for a metrics block.
